# Exploratory Data Analysis (EDA) Project

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a dataset using Python to uncover patterns, relationships, and insights. The analysis includes data cleaning, preprocessing, statistical summaries, and visualizations that help understand the structure and characteristics of the dataset.

---

## Objectives

- Understand the dataset structure
- Identify missing values and duplicates
- Perform data cleaning and preprocessing
- Analyze feature distributions
- Detect outliers
- Explore relationships between variables
- Generate meaningful visualizations
- Summarize key insights

---

## Dataset

Dataset: *Superstore-sales*

Description:
A structured dataset containing multiple features for exploratory analysis.

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Workflow

1. Import Libraries
